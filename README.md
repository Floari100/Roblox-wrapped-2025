# Roblox Wrapped 2025

A small “Wrapped”-style web app that generates a Roblox 2025 recap (slides with animations) from a Roblox username.

## Deployement
You can visit it on "https://roblox-wrapped-2025-6c9m.vercel.app"

## Features
- Swipe / click navigation (mobile-friendly)
- Animated slides (stats, streaks, player type, share card)
- Optional background music with Mute / Unmute
- Uses public Roblox endpoints (no API key)

## Requirements
- Node.js (LTS)

## Setup
```bash
npm install
npm run dev
